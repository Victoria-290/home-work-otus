# Проект: Simple Feedback API
REST API для сбора и хранения отзывов пользователей (о продукте, сайте, событии и т.п.).
Сначала данные сохраняются в JSON-файле, позже — в базе данных.
Можно получить список всех отзывов или конкретный отзыв по ID.

## MVP
POST /feedback — добавить отзыв
GET /feedbacks — получить список отзывов
GET /feedback/{id} — получить один отзыв
Все данные сохраняются в файл data/feedbacks.json
Валидация данных (например, текст не пустой, рейтинг от 1 до 5)
